# Лабораторной работа №1
## [Реализация RPC-сервиса с использованием gRPC](https://github.com/BratchenkoAA/1_lab_RS/tree/main/rasp_sistems)


В лабораторной работе реализован gRPC-сервис ProductCatalog для электронной коммерции, где сервер хранит каталог товаров в памяти, а клиент может запросить детальную информацию о конкретном товаре по его ID с помощью Unary RPC-метода GetProductDetails, получая в ответ название, описание, цену и наличие товара.

# Лабораторная работа № 2. 
## [Проектирование и реализация клиент-серверной системы. HTTP, веб-серверы и RESTful веб-сервисы](https://github.com/BratchenkoAA/ab_RS/tree/main/RS_2_lab)

В лабораторной работе №2 был разработан REST API "Библиотека книг" на Flask с управлением книгами через HTTP-методы (GET, POST, DELETE), проведен анализ HTTP-запросов к github.com с помощью curl, а также настроен веб-сервер Nginx в качестве обратного прокси для перенаправления запросов с порта 80 на Flask-приложение (порт 5000).
