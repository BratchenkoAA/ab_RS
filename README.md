# Лабораторной работа №1
## [Реализация RPC-сервиса с использованием gRPC](https://github.com/BratchenkoAA/1_lab_RS/tree/main/rasp_sistems)


В лабораторной работе реализован gRPC-сервис ProductCatalog для электронной коммерции, где сервер хранит каталог товаров в памяти, а клиент может запросить детальную информацию о конкретном товаре по его ID с помощью Unary RPC-метода GetProductDetails, получая в ответ название, описание, цену и наличие товара.
